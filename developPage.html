Construction/Electrical
In one of my technology courses, I was able to work hands on with many tools and equipments. I learned how to fabricate wood-working projects using construction machinery, learning how to properly dress lumber and effectively use machines. Throughout this experience I also reinforced proper safety standards in the woodshop. All of these hands-on projects helped me to gain confidence in using new tools and providing resources for ideas to come to life.
3D-Printed Arm
As a part of an ongoing Makers Club project, I have been building this 3D printed bionic arm which is controlled by the movements of a hand using the glove pictured. The electronics is based off of an Arduino Uno Microcontroller and resistence sensitive flex-sensors in order to detect movement. This information is then related to the corresponding servo motor which acts a pulley connected to each finger in order simulate similar hand movement.
To the right is an archived project, but I was still very proud to be apart of it! This was the first project in my Makerspace Club where I had to work with arts, mechanical, electrical, and coding. As you can probably tell, we were creating a life-sized BB8 robot--from scratch. This involved creating the exterior, building a inner motor chasis, and using a motor shield and Ardunio based circuit connected to a bluetooth module to control his movements using a remote control app.

I was nominated to represent my school at the Peel Science fair showcasing my experiment in measuring the energy outputs of convential fans versus my fan created to change speed based on surrounding temperatures. This complex circuit for which I made this schematic for, (if you would like the full project, let me know!) uses PWM (Pulse Width Modulation) through an NPN transistor and a DHT11 Humidity and Temperature sensor connected to an Arduino and 12V DC Fan. It is programmed to read the sensor and adjust the speed of the fan based on set ranges for the PWM.
